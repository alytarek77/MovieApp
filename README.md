# Movie Recommendation Web App

# Name:
Aly Abouzeid 

# Project Idea
This project is a movie web application that allows users to search for movies, view details, and manage a personal watchlist. It also includes an AI feature that recommends movies based on user preferences such as mood or genre.

# Description
The application is designed to demonstrate full stack development using a simple and user friendly interface. Users can interact with movie data and receive AI generated recommendations, making the app more interactive and useful.
